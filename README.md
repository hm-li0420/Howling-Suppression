## 背景
这是一个Python实现的啸叫抑制项目，整体项目分为三个部分：
1. 基于自适应法的啸叫抑制（adaptive_filter.py）
2. 基于啸叫检测和陷波器法的啸叫抑制（notch_filter.py）
3. 基于移频法的啸叫抑制（shift_freq.py）
